Measure - Any numeric column (with or without decimal) on which maths or stats operations can be performed to generate meaningful result iin business.
e.g. profit, tax, discount, quantity, cost, expense etc.  
Dimension - Any numeric or character or  date column which is not a measure and can be used to aggreagate measures 
e.g. department, name, address, city, category, birth date, join date, rating etc.

* Any primary or foreign key column is always considered as Dimension *

Only numberic operation allowed on dimension is COUNT
